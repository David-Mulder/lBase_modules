Assets

Styling and images are assets and can be placed in the /assets/ directories of the application or modules.

Module asset directories

After adding an asset directoy it's necessary to trigger the /admin/assets/regenerate page to regenerate the url rule file.

Asset preprocessors

Asset preprocessors are defined in the system itself and preprocess a certain type of file. This can be used to for example support .less (less.js) and .coffee (coffeescript) files. Default .less files are supported only. Practically this means you can drop .less files into your asset folders and treat it as if it's a css file.
