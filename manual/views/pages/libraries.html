Libraries

Libraries are stateless components defining a set of functions.

Using libraries

Libraries can be accessed using 

<cfset this.app.lib("libaryname").someFunction()>
or if the library is part of a module (provided the module is not registered, see registration) 
<cfset this.app.module("modulename").lib("libaryname").someFunction()>

Writing libraries

Please read the scoping guide to understand the absolute importance of making all variables local scoped.

Writing a library is as simple as dropping a component extending system.library into /application/libraries (or equivalent in a module): 

<cfcomponent extends="system.library">
	<cffunction name="someFunction">
		<cfset this.core.console.log("You accessed the function someFunction from inside a library")>
	</cffunction>
</cfcomponent>
